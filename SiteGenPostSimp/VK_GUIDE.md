# Как подключить ВКонтакте к проекту

## Что нам нужно

Для публикации постов в ВК нам нужны **два** секретных параметра:

1. **Access Token** — ключ доступа (длинная строка символов)
2. **ID сообщества** — номер вашего сообщества (только цифры)

---

## Шаг 1: Получить Access Token

### 1.1 Зайдите в настройки сообщества

1. Откройте ВКонтакте и зайдите в своё сообщество
2. Нажмите на **"три точки"** (⋮) или **"..."** под названием группы
3. Выберите **"Управление сообществом"**

### 1.2 Найдите раздел API

1. Если открылась новая версия интерфейса — **внизу страницы** есть ссылка **"Перейти к классической версии"** — нажмите её
2. В меню слева найдите пункт **"Работа с API"** и нажмите на него

### 1.3 Создайте ключ

1. Перейдите в раздел **"Управление сообществом"**
2. Нажмите на подраздел **"Access Tokens"**
3. Нажмите кнопку **"Создать ключ"**

### 1.4 Выберите права (разрешения)

**Обязательно поставьте галочки напротив:**
- ✅ `photos` — доступ к фото
- ✅ `stories` — доступ к историям
- ✅ `pages` — доступ к страницам
- ✅ `status` — доступ к статусу
- ✅ **`wall`** — публикация на стене (**ОБЯЗАТЕЛЬНО!**)
- ✅ **`offline`** — доступ когда вас нет онлайн (**ОБЯЗАТЕЛЬНО!**)

Если система попросит выбрать сообщество — выберите своё сообщество из списка.

### 1.5 Сохраните токен

**ВНИМАНИЕ! Токен показывается ТОЛЬКО ОДИН РАЗ сразу после создания!**

- Скопируйте токен **сразу** и сохраните в надёжное место
- Если вы закроете страницу — токен больше нельзя будет посмотреть
- Придётся создавать новый ключ

**Пример токена:**
```
vk1.a1234567890abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ_1234567890abcdefghijklmnopqrstuvwxyz1234567890
```

## Шаг 2: Узнать ID сообщества

ID сообщества — это **номер** в ссылке на вашу группу.

### Способ 1: По ссылке

Посмотрите на ссылку вашего сообщества:

| Ссылка | ID сообщества |
|--------|---------------|
| `vk.com/club123456789` | `123456789` |
| `vk.com/public123456789` | `123456789` |
| `vk.com/club_moyclub` | Нужен другой способ |

**Как найти ID:**
- Берём всё, что после `club` или `public`
- Убираем `club` или `public`
- Остаётся номер — это и есть ID

### Способ 2: Через поиск ВК

1. В поиске ВК введите `@название_группы` (без @)
2. В результатах поиска нажмите на сообщество
3. Посмотрите в адресную строку браузера — там будет номер после `club`

### Способ 3: Через код страницы

1. Откройте сообщество
2. Правой кнопкой → "Просмотр кода страницы"
3. Нажмите Ctrl+F
4. Введите `owner_id`
5. Найдите значение типа `"-123456789"` — это ID сообщества (число 123456789)

---

## Шаг 3: Внести данные в проект

### 3.1 Откройте файл config.py

Найдите файл `config.py` в папке проекта и откройте его текстовым редактором (Блокнот, VS Code и т.д.).

### 3.2 Заполните поля

Найдите эти строки и замените значения:

```python
# Token сообщества ВК (полученный на Шаге 1)
VK_ACCESS_TOKEN = "ВСТАВЬТЕ_ВАШ_ТОКЕН_СЮДА"

# ID сообщества (только цифры, без "club"!)
VK_GROUP_ID = "123456789"
```

**Пример заполненного файла:**
```python
VK_ACCESS_TOKEN = "vk1.aBcDeFgHiJkLmNoPqRsTuVwXyZ0123456789_AbCdEfGhIjKlMnOpQrStUvWxYz0123456789"
VK_GROUP_ID = "123456789"
```

### 3.3 Сохраните файл

Сохраните изменения в файле `config.py`.

---

## Шаг 4: Проверить подключение

### 4.1 Запустите сервер

Откройте терминал в папке проекта и выполните:
```bash
python app.py
```

### 4.2 Проверьте статус

1. Откройте в браузере: http://127.0.0.1:5004
2. Сгенерируйте любой пост
3. Под постом вы увидите блок "Публикация во ВКонтакте"

**Если ВК подключён:**
```
ВК подключен: Название вашей группы
[Опубликовать сейчас]
Запланировать: [___________] [Запланировать]
```

**Если ВК НЕ подключён:**
```
ВК не настроен
```

## Возможные ошибки

### Ошибка: "ВК не настроен"

**Причина:** Не заполнены VK_ACCESS_TOKEN или VK_GROUP_ID

**Решение:**
1. Проверьте файл `config.py`
2. Убедитесь, что токен и ID введены без пробелов и кавычек

### Ошибка: "Invalid access token"

**Причина:** Токен неверный или устарел

**Решение:**
1. Создайте новый токен в настройках сообщества
2. Обновите VK_ACCESS_TOKEN в config.py

### Ошибка: "User authorization failed"

**Причина:** Не выданы все нужные права

**Решение:**
1. Зайдите в настройки сообщества
2. "Работа с API" → "Access Tokens"
3. Создайте новый ключ
4. Обязательно поставьте галочки на `wall` и `offline`

### Ошибка: "Permission denied"

**Причина:** У вас нет прав на публикацию в этом сообществе

**Решение:**
1. Убедитесь, что вы создали токен именно для этого сообщества
2. Проверьте, что вы являетесь администратором сообщества

| Что нужно | Где взять | Пример |
|-----------|-----------|--------|
| Access Token | Управление сообществом → Работа с API → Access Tokens → Создать ключ | `vk1.aBcDeFgHiJk...` |
| Group ID | Ссылка на группу: `vk.com/club123456789` | `123456789` |

---

## Важные правила

1. **Токен — секрет!** Не показывайте его никому
2. **Право `wall`** — обязательно для публикации постов
3. **Право `offline`** — позволяет публиковать когда вас нет в сети
4. **Не спамьте!** ВК может заблокировать сообщество за частые публикации
5. **Лимит постов** — не публикуйте более 50 постов в сутки

---

## Если что-то не получается

1. Проверьте, что токен скопирован полностью (нет пробелов в начале/конце)
2. Проверьте, что ID — только цифры (без "club", "public" и других букв)
3. Перезапустите сервер после изменений в config.py
4. Попробуйте создать новый токен
