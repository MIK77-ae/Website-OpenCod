# Генератор постов для товаров

Flask-сайт для создания уникальных продающих постов с помощью языковой модели (GPT) через ProxyAPI.

## Возможности

- **Генерация постов с ИИ** — GPT-модель создаёт оригинальные посты
- **4 тона текста** — дружелюбный, восторженный, деловой, спокойный
- **Готовые хэштеги и CTA** — модель добавляет релевантные теги и призыв к действию
- **Копирование в один клик** — готовый пост копируется в буфер обмена
- **Публикация в ВК** — опубликуйте пост или запланируйте на время
- **Избранное** — сохраняйте посты для последующего использования
- **Отложенные посты** — запланируйте публикацию на нужное время
- **Без базы данных** — простой код для понимания

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сервер

**Windows (рекомендуется):**
Двойной клик по файлу `run.bat`

**Или вручную:**
```bash
python run.bat
```

**Linux/Mac:**
```bash
export PROXY_API_KEY=ваш-ключ
export VK_ACCESS_TOKEN=ваш-токен
export VK_GROUP_ID=123456789
python app.py
```

### 3. Откройте в браузере

```
http://127.0.0.1:5004
```

## НАСТРОЙКА КЛЮЧЕЙ

### Все ключи хранятся в переменных окружения

Это нужно для безопасности — ключи не попадают в GitHub.

### Windows

Отредактируйте файл `run.bat`, изменив значения:

```batch
set PROXY_API_KEY=ваш-прокси-ключ
set VK_ACCESS_TOKEN=ваш-вк-токен
set VK_GROUP_ID=123456789
```

### Linux/Mac

```bash
export PROXY_API_KEY=ваш-прокси-ключ
export VK_ACCESS_TOKEN=ваш-вк-токен
export VK_GROUP_ID=123456789
```

## ПУБЛИКАЦИЯ ВО ВКОНТАКТЕ

### Как получить токен и ID сообщества

Подробная инструкция в файле **VK_GUIDE.md**

### Кратко:

1. **Токен:** Управление сообществом → Работа с API → Access Tokens → Создать ключ
   - Обязательно: `wall` и `offline`
   - Сохраните сразу — показывается один раз

2. **ID сообщества:** Из ссылки `vk.com/club123456789` → `123456789`

## ИНТЕРФЕЙС

### Вкладка "Генератор"
- Поля: название, описание*, цена, тон текста
- Кнопка "Сформировать пост"
- Под постом:
  - Кнопки: Скопировать, В избранное
  - Блок ВК: Опубликовать сейчас / Запланировать

### Вкладка "Избранное"
- Список сохранённых постов
- Кнопки: Использовать, Удалить

### Вкладка "Отложенные"
- Список запланированных постов
- Кнопка: Удалить
