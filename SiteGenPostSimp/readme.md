# Генератор постов для товаров

Flask-сайт для создания уникальных продающих постов с помощью языковой модели (GPT) через ProxyAPI.

## Возможности

- **Генерация постов с ИИ** —GPT-модель создаёт оригинальные посты
- **4 тона текста** — дружелюбный, восторженный, деловой, спокойный
- **Готовые хэштеги и CTA** — модель добавляет релевантные теги и призыв к действию
- **Копирование в один клик** — готовый пост копируется в буфер обмена
- **Без базы данных** — простой код для понимания

## Структура проекта

```
SiteGenPostSimp/
├── app.py              # Flask-приложение (маршруты)
├── api_client.py      # Клиент для ProxyAPI

├── voice.md           # Настройки стиля и промптов
├── run.bat            # Запуск сервера (Windows)
├── requirements.txt   # Зависимости Python
├── templates/
│   └── index.html     # HTML-шаблон страницы
└── static/
    └── style.css      # Стили CSS
```

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите сервер:**
   ```bash
   python app.py
   ```
   Или двойным кликом по файлу `run.bat`

3. **Откройте в браузере:**
   ```
   http://127.0.0.1:5004
   ```

## Настройка API

API-ключ ProxyAPI настраивается в файле `config.py`:

```python
PROXY_API_KEY = "ваш-ключ"
MODEL_NAME = "gpt-5.4-mini"
```

Получить ключ: [https://proxyapi.ru/keys](https://proxyapi.ru/keys)

## Тоны текста

| Тон | Описание | Когда использовать |
|-----|---------|-------------------|
| Дружелюбный 😊 | Тёплый, располагающий | Повседневные товары, блог |
| Восторженный 🔥 | Эмоциональный, энергичный | Новинки, акции |
| Деловой 📊 | Уверенный, профессиональный | B2B, техника |
| Спокойный 🌿 | Размееренный, мягкий | Услуги, красота |

## Поля формы

- **Название товара** — необязательное
- **Описание товара** — обязательное (*)
- **Цена или выгода** — необязательное
- **Тон текста** — выбор из 4 вариантов

## Кастомизация стиля

Все настройки в файле `voice.md`:

- `TONES` — определения тонов
- `SYSTEM_PROMPT_TEMPLATE` — промпт для ИИ
- `HASHTAG_RULES` — правила хэштегов
- `CTA_RULES` — варианты CTA
- `AVOID` — слова для избегания

## Troubleshooting

**ERR_CONNECTION_REFUSED**
- Запустите сервер: `python app.py`
- Проверьте, что порт 5004 свободен

**Ошибка API**
- Проверьте ключ в `config.py`
- Проверьте баланс в личном кабинете ProxyAPI

**Пустой ответ**
- Измените `TEMPERATURE` в `config.py`
- Убедитесь, что описание товара информативное
