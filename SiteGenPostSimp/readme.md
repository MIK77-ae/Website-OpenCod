# Генератор постов для товаров

Простой Flask-сайт для создания уникальных продающих постов с помощью языковой модели через ProxyAPI.

## Возможности

- **Генерация постов с ИИ** — посты создаются с помощью GPT-модели через ProxyAPI
- **4 тона текста** — дружелюбный, восторженный, деловой, спокойный
- **Готовые хэштеги и CTA** — модель добавляет релевантные теги и призыв к действию
- **Копирование в один клик** — готовый пост легко скопировать в буфер обмена
- **Без базы данных** — простой код, понятный для новичка

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите сервер:**
   ```bash
   python app.py
   ```

3. **Откройте в браузере:**
   ```
http://127.0.0.1:5004
   ```

## Настройка API

API-ключ ProxyAPI настраивается в файле `config.py`:

```python
PROXY_API_KEY = "ваш-ключ"
MODEL_NAME = "gpt-5.4-mini"  # рекомендуемая модель
```

Получить ключ: [https://proxyapi.ru/keys](https://proxyapi.ru/keys)

## Тоны текста

| Тон | Описание | Когда использовать |
|-----|---------|-------------------|
| Дружелюбный 😊 | Тёплый, располагающий | Повседневные товары, блог |
| Восторженный 🔥 | Эмоциональный, энергичный | Новинки, акции |
| Деловой 📊 | Уверенный, профессиональный | B2B, техника |
| Спокойный 🌿 | Размееренный, мягкий | Услуги, услуги красоты |

## Поля формы

- **Название товара** — необязательное поле
- **Описание товара** — обязательное поле (отмечено *)
- **Цена или выгода** — необязательное поле
- **Тон текста** — выбор стиля поста

## Кастомизация стиля

Все настройки генерации находятся в файле `voice.md`:

- `TONES` — определения тонов и их параметров
- `SYSTEM_PROMPT_TEMPLATE` — промпт для ИИ
- `HASHTAG_RULES` — правила хэштегов
- `CTA_RULES` — варианты CTA
- `AVOID` — слова и фразы, которых следует избегать

## Troubleshooting

**Ошибка "Invalid API key"**
- Проверьте ключ в `config.py`
- Убедитесь, что ключ активен в личном кабинете ProxyAPI

**Ошибка "Timeout"**
- Проверьте интернет-соединение
- Увеличьте `API_TIMEOUT` в `config.py`

**Пустой ответ от модели**
- Попробуйте изменить `TEMPERATURE` в `config.py`
- Убедитесь, что описание товара информативное
