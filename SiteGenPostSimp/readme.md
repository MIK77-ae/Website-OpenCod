# Генератор постов для товаров

Flask-сайт для создания уникальных продающих постов с помощью языковой модели (GPT) через ProxyAPI.

## Возможности

- **Генерация постов с ИИ** — GPT-модель создаёт оригинальные посты
- **4 тона текста** — дружелюбный, восторженный, деловой, спокойный
- **Готовые хэштеги и CTA** — модель добавляет релевантные теги и призыв к действию
- **Копирование в один клик** — готовый пост копируется в буфер обмена
- **Публикация в ВК** — опубликуйте пост или запланируйте на время
- **Избранное** — сохраняйте посты для последующего использования
- **Без базы данных** — простой код для понимания

## Установка и запуск

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите сервер:**
   ```bash
   python app.py
   ```
   Или двойным кликом по файлу `run.bat`

3. **Откройте в браузере:**
   ```
   http://127.0.0.1:5004
   ```

## ПУБЛИКАЦИЯ ВО ВКОНТАКТЕ

### Как подключить ВК — пошаговая инструкция

#### Шаг 1: Получите токен сообщества

1. **Откройте настройки сообщества:**
   - Зайдите в своё сообщество во ВКонтакте
   - Нажмите на три точки «...» → «Управление сообществом»
   - Если открылась новая версия — нажмите «Перейти к классической версии» (снизу)
   - В меню слева выберите «Работа с API»

2. **Создайте ключ доступа:**
   - Раздел «Управление сообществом»
   - Подраздел «Access Tokens»
   - Нажмите кнопку «Создать ключ»
   - Поставьте галочки напротив разрешений:
     - ✅ `photos`
     - ✅ `stories`
     - ✅ `pages`
     - ✅ `status`
     - ✅ **`wall`** (обязательно!)
     - ✅ **`offline`** (обязательно!)
   - Если система попросит выбрать сообщество — выберите своё
   - Нажмите «Создать» и **сразу скопируйте полученный токен** (он показывается только один раз!)

3. **Сохраните токен в секрете** — не показывайте его никому!

#### Шаг 2: Узнайте ID сообщества

ID сообщества — это номер, который идёт после «club» или «public» в ссылке.

**Способ 1:** Смотрите ссылку на сообщество:
- Если ссылка: `vk.com/club123456789` → ID: `123456789`
- Если ссылка: `vk.com/public123456789` → ID: `123456789`

**Способ 2:** Если у сообщества короткое имя (например, `vk.com/moyclub`):
- Откройте сообщество
- Посмотрите в адресной строке браузера — там должен быть цифровой ID
- Или: нажмите правой кнопкой на аватарку → «Копировать ссылку» → посмотрите номер после `club`

**Способ 3:** Через поиск ВК:
- Напишите в поиске `@short_name` (где short_name — короткое имя после `vk.com/`)
- В результатах будет видно цифровой ID

#### Шаг 3: Внесите данные в проект

Откройте файл `config.py` и добавьте:

```python
# Токен сообщества ВК (полученный на шаге 1)
VK_ACCESS_TOKEN = "ваш-токен-здесь"

# ID сообщества (только цифры, без "club"!)
VK_GROUP_ID = "123456789"
```

#### Шаг 4: Проверьте подключение

1. Запустите сервер: `python app.py`
2. Откройте http://127.0.0.1:5004
3. В блоке «Опубликовать» должно появиться: **«ВК подключен: [Название вашего сообщества]»**

Если видите «ВК не настроен» — проверьте, что токен и ID введены правильно.

### Что можно делать с ВК

| Функция | Описание |
|---------|----------|
| **Опубликовать сейчас** | Пост выходит сразу |
| **Запланировать** | Выберите дату и время — пост выйдет автоматически |
| **Удалить запланированный** | Отменить публикацию можно в любой момент |

### Важные правила

1. **Токен одноразовый** — сохраните его сразу после получения, посмотреть его снова нельзя
2. **Право «offline»** — позволяет публиковать посты даже когда вы не заходите в ВК
3. **Право «wall»** — обязательно для публикации на стене сообщества
4. **Не публикуйте спам** — ВК может заблокировать сообщество за нарушение правил

## Избранное

Сохраняйте посты в избранное для:

- Быстрого доступа без повторной генерации
- Сравнения разных вариантов
- Переиспользования в будущем

Посты сохраняются в файл `favorites.json`.

## Отложенные посты

Запланируйте публикацию заранее:

1. Сгенерируйте пост
2. Нажмите «Опубликовать»
3. Выберите «Запланировать на время»
4. Укажите нужную дату и время
5. Пост выйдет автоматически в указанное время

Пооты сохраняются в файл `scheduled_posts.json`.

## Настройка ProxyAPI

API-ключ и модель настраиваются в файле `config.py`:

```python
PROXY_API_KEY = "ваш-ключ"
MODEL_NAME = "gpt-5.4-mini"
```
