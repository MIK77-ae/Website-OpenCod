# VibeCode — Курс по вайб-кодингу

Конверсионный лендинг для продажи онлайн-курса по вайб-кодингу.

## О проекте

Современный, стильный сайт с тёмной темой и плавными анимациями. Разработан с целью привлечения и конверсии пользователей в студентов курса.

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, анимации)
- JavaScript (vanilla)

## Структура

```
CloudCodSite/
├── index.html   # Главная страница
├── style.css    # Стили
├── script.js    # Интерактивность
└── README.md    # Этот файл
```

## Запуск

Откройте `index.html` в любом современном браузере.

## Особенности

- Адаптивный дизайн (мобильные + десктоп)
- Плавные анимации при скролле
- Интерактивные элементы (FAQ-аккордеон, hover-эффекты)
- Живой фон с градиентными орбами

## Цветовая схема

- Основной: `#1E1E2F`
- Акцент персиковый: `#FF6F61`
- Акцент синий: `#00BFFF`
- Фон: `#121212`
