# Настройки стиля постов

## Общий тон постов

**Варианты:**
- `friendly` (дружелюбный) - тёплый, располагающий, для широкой аудитории
- `energetic` (энергичный) - динамичный, мотивирующий, для активных людей
- `professional` (профессиональный) - деловой, для бизнес-аудитории
- `luxury` (премиум) - элегантный, для дорогих товаров

**Рекомендация:** Для большинства товаров подходит `friendly` - он универсален и хорошо воспринимается.

---

## Длина поста

**Варианты:**
- `short` (короткий) - 2-3 предложения, для соцсетей с ограничением
- `medium` (средний) - 4-6 предложений, оптимально для большинства платформ
- `long` (длинный) - 7-10 предложений, для детального описания

**Рекомендация:** `medium` - самый универсальный вариант.

---

## Количество эмодзи

**Рекомендация:** 1-3 эмодзи на пост.

- 1 эмодзи - лаконично, для делового стиля
- 2 эмодзи - баланс, рекомендуется
- 3 эмодзи - ярко, для дружелюбного стиля

**Важно:** Не перегружайте пост эмодзи - это выглядит непрофессионально.

---

## Стиль заголовков

**Варианты:**
- `question` (вопрос) - привлекает внимание, вызывает интерес
  - Пример: "Хотите узнать про новинку?"
- `statement` (утверждение) - информативный, для фактов
  - Пример: "Представляем новый товар!"
- `benefit` (выгода) - продающий, акцент на выгоде
  - Пример: "Сэкономьте 30% с этим товаром"

**Рекомендация:** `benefit` - самый продающий вариант.

---

## Стиль CTA (призыва к действию)

**Варианты:**
- `soft` (мягкий) - ненавязчивый, для внимательной аудитории
  - Пример: "Хотите узнать больше? Пишите в комментариях!"
- `strong` (активный) - побуждает к действию
  - Пример: "Заказывайте прямо сейчас!"
- `curious` (интригующий) - вызывает любопытство
  - Пример: "А вы уже пробовали?"

**Рекомендация:** `soft` - не отпугивает, но мотивирует.

---

## Правила для хэштегов

**Количество:** 3-7 хэштегов оптимально.

**Типы хэштегов:**
- Название товара: `#товар_продажа`, `#товар_купить`
- Общие: `#покупки`, `#скидка`, `#акция`, `#новинка`, `#рекомендую`
- Выгоды: `#выгода`, `#качество`, `#特惠` (китайский "спецпредложение")
- Международные: `#sale`, `#deal`, `#buy`

**Важно:**
- Используйте 1-2 хэштега с названием товара
- Добавьте 2-3 общих тематических хэштега
- Не используйте более 10 хэштегов - это спам

---

## Чего избегать

### ❌ Не делайте:
- **Много капитаized текста** - ВЫГЛЯДИТ КАК КРИК
- **Чрезмерное количество эмодзи** - читается как спам
- **Длинные стены текста** - разбивайте на абзацы
- **Ложные обещания** - не обещайте того, чего нет
- **Отрицание конкурентов** - не опускайте других
- **Много хэштегов** - больше 10 - это спам
- **Слишком формальный язык** - теряется контакт с аудиторией

### ✅ Делайте:
- Пишите короткими предложениями
- Используйте активные глаголы
- Делайте акцент на выгоде для клиента
- Добавляйте социальные доказательства
- Завершайте чётким призывом к действию

## Как изменить настройки

В файле `app.py` найдите секцию "НАСТРОЙКИ ГЕНЕРАТОРА" в начале файла.
Там можно изменить:
- `POST_TONE` - общий тон
- `POST_LENGTH` - длина
- `EMOJI_COUNT` - количество эмодзи
- `TITLE_STYLE` - стиль заголовка
- `CTA_STYLE` - стиль призыва
- `HASHTAG_COUNT` - количество хэштегов

Всё просто - меняйте значения и запускайте заново!
