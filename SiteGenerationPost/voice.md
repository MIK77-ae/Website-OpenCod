# Настройки стиля постов (версия с ИИ)

## Общее

Генератор использует языковую модель gpt-5.4-mini через ProxyAPI для создания уникальных постов.

## Доступные настроения

| Настроение | Описание | Эмодзи |
|------------|----------|--------|
| friendly | Дружелюбный, теплый | 😊 |
| energetic | Энергичный, динамичный | 🔥 |
| professional | Профессиональный, деловой | 💼 |
| luxury | Премиум, элегантный | ✨ |
| humor | С юмором, веселый | 😄 |
| curious | Интригующий, загадочный | 🤔 |
| inspirational | Вдохновляющий, мотивирующий | 🌟 |

## Параметры модели

В файле `app.py` можно настроить:

```python
MODEL = "gpt-5.4-mini"  # Модель (см. ниже)
temperature = 0.9       # Креативность (0-2)
max_tokens = 1500        # Макс. длина ответа
```

## Доступные модели

| Модель | Ввод (₽/1М) | Вывод (₽/1М) |
|--------|-------------|--------------|
| gpt-5.4-nano | 61 | 380 |
| gpt-5.4-mini | 230 | 1 370 |
| gpt-5.4 | 760 | 4 550 |
| gpt-5.5 | 1 520 | 9 100 |
| gpt-5.4-pro | 4 740 | 28 400 |

**Рекомендация:** gpt-5.4-mini - оптимальное соотношение цена/качество.

## Как изменить модель

В файле `app.py` найдите строку:

```python
MODEL = "gpt-5.4-mini"
```

И замените на другую модель из списка выше.

## Что генерирует ИИ

Модель создает пост с:
- Уникальным заголовком (не шаблонным)
- Описанием товара на основе ссылки
- Выгодой для покупателя
- Хэштегами (3-7 штук)
- CTA (призыв к действию)
- Разными эмоциями в зависимости от выбранного настроения
