# Генератор постов для товаров

Простой и понятный генератор маркетинговых постов на Flask.

## Быстрый старт

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сайт

```bash
python app.py
```

### 3. Откройте в браузере

Перейдите по адресу: http://127.0.0.1:5001

## Как это работает

1. **Заполните форму** - название товара, описание, цена/выгода, настроение
2. **Нажмите "Сформировать пост"** - получите готовый текст
3. **Скопируйте пост** - нажмите кнопку, пост скопируется в буфер обмена

## Структура проекта

```
SiteGenerationPost/
├── app.py           # Основной код Flask-приложения
├── templates/
│   └── index.html   # HTML-страница
├── voice.md         # Настройки стиля постов
└── requirements.txt # Зависимости
```

## Настройка стиля

Откройте `app.py` и измените настройки в начале файла:

```python
POST_TONE = "friendly"      # friendly, energetic, professional, luxury
POST_LENGTH = "medium"      # short, medium, long
EMOJI_COUNT = 2            # 1-3
TITLE_STYLE = "benefit"     # question, statement, benefit
CTA_STYLE = "soft"         # soft, strong, curious
HASHTAG_COUNT = 5          # количество хэштегов
```

Подробности в файле `voice.md`.

## Без нейросетей и базы данных

Генератор работает на основе шаблонов - простой и понятный код для новичка.

## Требования

- Python 3.8+
- Flask 3.0.0
