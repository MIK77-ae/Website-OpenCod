# Генератор постов для товаров (с ИИ)

Генератор маркетинговых постов с использованием языковой модели ProxyAPI.

## Быстрый старт

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сайт

```bash
python app.py
```

### 3. Откройте в браузере

Перейдите по адресу: http://192.168.1.14:5002

## Как это работает

1. **Вставьте ссылку на товар** - ИИ проанализирует страницу
2. **Выберите настроение** - дружелюбное, энергичное, юмор и т.д.
3. **Нажмите "Создать пост с ИИ"** - получите уникальный текст
4. **Скопируйте пост** - готовый маркетинговый текст

## Настройки ProxyAPI

В файле `app.py` можно изменить:

- `PROXY_API_KEY` - ваш ключ API
- `MODEL` - модель для генерации (gpt-5.4-mini, gpt-5.4, gpt-5.5 и др.)

## Требования

- Python 3.8+
- Flask 3.0.0
- requests

## Стоимость

Модель gpt-5.4-mini:
- Ввод: 230 ₽ за 1 млн токенов
- Вывод: 1 370 ₽ за 1 млн токенов

Один пост ~50-100 токенов = копейки!
