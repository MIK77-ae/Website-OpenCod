# Генератор постов для товаров (с ИИ + Selenium)

Генератор маркетинговых постов с использованием языковой модели ProxyAPI и автоматическим чтением ссылок.

## Быстрый старт

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустите сайт

```bash
python app.py
```

### 3. Откройте в браузере

**http://192.168.1.14:5003**

## Как это работает

1. **Вставьте ссылку на товар** - ИИ автоматически прочитает данные с страницы
2. **Выберите настроение** - дружелюбное, энергичное, юмор и т.д.
3. **Нажмите "Создать пост с ИИ"** - получите уникальный текст
4. **Скопируйте пост** - готовый маркетинговый текст

## Поддерживаемые сайты

- ✅ Ozon
- ✅ Яндекс.Маркет
- ✅ Wildberries
- ✅ Другие интернет-магазины

## Оптимизация

Первый запрос занимает чуть дольше (запуск браузера), последующие - быстрее благодаря переиспользованию браузера.

## Требования

- Python 3.8+
- Chrome browser
- Flask 3.0.0
- Selenium 4.43.0
