# AI-Дайджест для новичков

Карточки которые учат работе с ИИ

## Описание

Веб-приложение на Flask с интерактивными карточками для изучения искусственного интеллекта. Идеально для новичков, которые хотят разобраться в основах нейросетей, промптинга и первых проектах.

## Возможности

- 📚 11 информативных карточек
- 🏷️ Фильтрация по тегам
- 🌙 Современный тёмный дизайн
- 📱 Адаптивность (ПК и телефон)
- ✨ Плавные анимации

## Структура проекта

```
Flask/
├── app.py           # Flask-приложение
├── content.json    # Данные карточек
├── readme.md       # Этот файл
├── templates/
│   └── index.html  # Главная страница
└── static/
    ├── css/
    │   └── style.css  # Стили
    └── js/
        └── script.js   # Логика
```

## Запуск

```bash
cd Flask
pip install -r requirements.txt
python app.py
```

Открой в браузере: http://127.0.0.1:5001

## Темы карточек

- Нейросети и LLM
- Промптинг
- Первые проекты
- Cloud Code
- ИИ агенты и мультиагентные системы
